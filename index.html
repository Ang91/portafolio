Sobre Mí

Soy desarrolladora frontend con experiencia en la creación de interfaces web atractivas y funcionales utilizando HTML, CSS y JavaScript. Mi enfoque está en proporcionar una experiencia de usuario intuitiva y accesible, aplicando principios de diseño UX/UI en cada proyecto. Me apasiona aprender y resolver problemas de manera eficiente, y busco siempre mejorar mis habilidades mediante la colaboración y el aprendizaje continuo. Mi objetivo es ayudar a las empresas a crear productos web que no solo sean visualmente atractivos, sino también fáciles de usar y optimizados para el rendimiento.
